\velkynadpis{The Same, Is It?}

There are two newline-separated integers on the input. If these numbers are equal, print a single line with the text \texttt{Hura, rovnaju sa.}. If the numbers are different, print \textt{Nerovnaju sa.}.

\nadpis{Examples}
\vstup
47
47
\vystup
Hura, rovnaju sa.
\koniec
