\velkynadpis{The Same, Is It?}

There are two space-separated integers on the input. If these numbers are equal, print a single line with the text \texttt{Hura}. If the numbers are different, print nothing.

\nadpis{Examples}
\vstup
47 47
\vystup
Hura
